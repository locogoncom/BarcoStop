# Google Play Console Assets - BarcoStop

## Archivos listos

- `icon-512.png` -> App icon (Google Play)
- `feature-graphic-1024x500.png` -> Feature graphic
- `icon-source.svg` -> Fuente vectorial del icono
- `feature-graphic-source.svg` -> Fuente vectorial de la feature graphic

## Requisitos de Play Console

- App icon:
  - PNG
  - 512 x 512 px
  - 32-bit
  - fondo solido (sin pixelado)
- Feature graphic:
  - PNG o JPG
  - 1024 x 500 px

## Ubicacion

`public/assets/google-play/`

## Nota

Si quieres ajustes de estilo (mas minimalista, mas premium, colores, barco/ola/anchor), se modifica primero el `.svg` y se vuelve a exportar PNG.
